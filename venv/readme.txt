BRIEF INSTRUCTIONS ON HOW TO MAKE THIS APP RUN
To install in a virtual environment (recommended):
Python 3.x required.

Unix/MacOS:
python3 -m venv venv
source venv/bin/activate
pip install h2o-wave

Windows:
python -m venv venv
.\venv\Scripts\activate
pip install h2o-wave

Run the main script.
wave main.py.

This is a work in progress.
